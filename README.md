#calculator
Simple Python cli based Calculator
A straight forward calculator built in Python for performing basic arithmetic operations directly in the command line.
Features
1.Addition(+)
2.Subtraction(-)
3.Multiplication(*)
4.Division(/)
5.Robust handling for division by zero errors.
6.Input validation to ensure both inputs are valid numbers.
7.Option to perform multiple calculations in a single session.

#Usage
You must have Python 3 installed on your system.
Running the Calculator
Save the code from calculator.py to a file name of your choice
Open your command line
Run the script
